# DOMNamespaces::react-dom

## 概述

html节点命名空间的集合。

## 源码

    'use strict';
    
    var DOMNamespaces = {
      html: 'http://www.w3.org/1999/xhtml',
      mathml: 'http://www.w3.org/1998/Math/MathML',
      svg: 'http://www.w3.org/2000/svg'
    };
    
    // 节点命名空间的集合
    module.exports = DOMNamespaces;
